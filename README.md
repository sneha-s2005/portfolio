## 🎨 Features & Highlights

- **✨ Glassmorphism UI**: Beautiful, premium frosted glass elements built with modern CSS effects and Tailwind CSS.
- **🌓 Dark Mode**: Toggle between light and dark modes with a smooth transition.
- **🚀 Framer Motion Animations**: Micro-interactions, timeline entrances, and smooth scrolling for a premium UX/UI feel.
- **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile displays.
- **📂 Detailed Sections**:
  - **Hero**: Eye-catching introduction with typing effects.
  - **About Me**: Brief background of my career goals and highlights.
  - **Skills**: Visual meters showcasing technical proficiency, tools, and soft skills.
  - **Education**: B.Tech Computer Science Engineering timeline.
  - **Projects**: Card-based showcase of featured applications with source code and live links.
  - **Hackathons**: Milestone timeline of participations, roles, and awards.
  - **Certifications**: List of technical accomplishments and courses completed.
  - **Contact**: Interactive contact form with direct mail and phone links.

## 🛠️ Installation & Local Setup

To run this project locally, follow these simple steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/sneha-s2005/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Start Development Server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to view the application.

4. **Build for Production:**
   ```bash
   npm run build
   ```
